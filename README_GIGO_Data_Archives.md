## 🔐 What Is Cryptographic Hashing?

**Hashing** turns a file into a unique string of characters — like a fingerprint.  
If even one byte of the file changes, the hash is completely different.  

We use the **SHA-256** algorithm — a global standard used in security, blockchain, and compliance industries.  

### This ensures:
- ✅ **Tamper-evidence**  
   Anyone can check the hash to confirm the file hasn’t changed  
   → Because the same file will always produce the same hash  

- ✅ **Authenticity**  
   We know when the file was made and what its exact contents were  
   → Because the hash log is timestamped and committed  

- ✅ **Audit Readiness**  
   Investors, regulators, or future teams can validate history  
   → Without relying on belief or memory
