## 🔐 What Is Cryptographic Hashing?

**Hashing** turns a file into a unique string of characters — like a digital fingerprint.  
If even one byte of the file changes, the hash is completely different.

We use the **SHA-256** algorithm — a global standard used in security, blockchain, and compliance industries.

### This ensures:

- ✅ **Tamper-evidence**  
   Anyone can verify that a file hasn’t changed  
   → Because the same file always creates the same hash  

- ✅ **Authenticity**  
   Timestamped and fingerprinted at creation  
   → We know exactly what was written and when  

- ✅ **Audit readiness**  
   Enables due diligence, legal protection, and historical confidence  
   → With no need to “trust” the human who archived it
